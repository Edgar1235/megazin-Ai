# MARKET OS v2.5 — Інтелектуальна екосистема покупок

### 🚀 Мета проєкту (Elevator Pitch)
Market OS — це концептуальна веб-операційна система з інтегрованим ШІ-агентом, яка перетворює звичайний онлайн-шопінг на інтерактивний досвід. Мета проєкту — продемонструвати синергію сучасного UI/UX дизайну та великих мовних моделей (LLM) для автоматизації консультування та управління кошиком.

### 👥 Цільова аудиторія
- Користувачі, які цінують футуристичні інтерфейси.
- Покупці, яким потрібна допомога персонального асистента в реальному часі.
- Розробники, що досліджують інтеграцію ШІ у веб-додатки.

### 🧠 Використані ШІ-паттерни
1. **AI Agent (Contextual Memory):** Бот зберігає історію переписки, що дозволяє вести складні діалоги.
2. **System Role Prompting:** ШІ чітко виконує роль менеджера магазину з доступом до внутрішньої бази цін.
3. **Error Handling & Resilience:** Реалізовано механізм захисту від збоїв API (помилка 503), що забезпечує стабільність UI.

### 🛠 Інструкція із запуску
1. Скопіюйте всі файли проєкту на ваш комп'ютер.
2. Переконайтеся, що у файлі `мозок.js` вказано ваш діючий API ключ від Google Gemini.
3. Відкрийте файл `головна робочий стіл.html` у будь-якому сучасному браузері (рекомендовано Safari або Chrome на Mac).
4. **Важливо:** Для роботи ШІ необхідне активне підключення до інтернету.

### 🏗 Технологічний стек
- **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (ES6+).
- **Пам'ять:** LocalStorage для збереження кошика та балансу.
- **ШІ:** Google Gemini API.
